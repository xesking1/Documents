Introducción

¿Qué és html?

HTML (HyperText Markup Language) és el lenguaje de marcas para crear páginas web. Define la estructura de la página con etiquetas que el navegador interpreta para mostrar el contenido

Estructura Básica de un HTML

Estructura general

Un documento HTML es una estructura jerarquica de etiquetas que define diferentes partes de una página. Cada pagina HTML sigue una estructura similar el principio empieza por "!DOCTYPE html", seguido de las etiquetas "html","head","body".

Etiquetas básicas

  • title

    Esta etiqueta define el titulo que aparece en la pestaña del navegador o los resultados del buscador.

  • h1 a h6

    Utilizados para definir bloques de texto o paragrafos.

  • p

    Se utiliza para definir bloques de texto o paragrafos.

  • br

    Fuerza al texto a comenzar una nueva linea.

  • & nbsp

    Inserta un espacio en blanco que no se rompe entre lineas

Elementos html

Diferencia entre etiquetas y elementos

Etiqueta (Tag):

Parte que define el inicio i el final de un componente en html. Se delimita con los simbolos "<" ">".

Elemento

Un elemento incluye la etiqueta de abertura, el contenido, i la etiqueta de cierre. Los elementos pueden estar vacios o contener otros elementos dentro.

Elementos HTML y anidamiento

Elementos anidados

El anidmiento se refiere a colocar un elemento HTML dentro de otro elemento. Es importante seguir correctamente el orden de las etiquetas: la primera que se abre es la ultima que se cierra.

Atributos HTML

Los atributos son elementos que añaden informacion extra a las etiquetas HTML para modificar el comportamiento o la apariencia. Siempre se colocan dentro de la etiqueta de abertura y siguen el formato nombre="valor".

Texto formatado

  • b y strong: Negrita

    b: Aplica negrita sin indicar ninguna importancia semantica.

    strong: Aplica negrit y indica que el texto es importante.

  • i y em: cursiva

    i: Aplica cursiva sin implicar émfasi.

    em: Aplica cursiva con énfasi, indica que el texto tiene importancia.

  • u: subrayado

    Se utiliza para subrayar el texto

  • strike o s: texto tachado

    Aplica una raya negra en el texto, indica que esta borrada o ya no es relevante.

  • sup: superíndex

    Se utiliza para elevar el texto, como en componentes matematicos

  • sub: subíndex

    Se utiliza para poner texto en la posicio de subíndex, utilizado en formulas quimicas.

Comentarios

Los comentarios son lineas de texto que se añaden al codigo para dejar notas o explicaciones, pero que no son visibles para los usuarios finales en la pagina web.

Se utilizan para ayudar los desenvolupadores a documentar el codigo, describir secciones especificas o dejar indicaciones para futuros desenvolupadores.

No afectan al rendimiento ni a la visualizacionde la pagina del navegador.

Buenas practicas

Manten los claros y concisos, evitar comentarios excesivos y no dejar comentarios innecesarios en produccion.

Insercion de imagenes

  • La etiqueta img se utilizaa para insertar imagenes en una pagina web.

  • No tiene etiqueta de cierre.

  • Es esencial proporionar informacion sobre la imagen para que se muestre correctamente y para mejorar la accesibilidad.

La etiqueta img se utiliza para insertar immagenes en una pagina web. No tiene etiqueta de cierre. Es esencial proporcionar informacion sobre la imagen para que se muestre correctamente y para mejorar la accesibilidad

Atributos de imagen

  1. src (source)

    Define la fuente/camino de la imagen. Puede ser un camino relativo o una URL completa.

  2. alt (alternate text)

    Proporciona un texto alternativo que se muestra si la imagen no se carga o no es leida por lectores de pantalla. Es fundamental para la accesibilidad y SEO

  3. width y height

    Especifica el ancho (width) y el (alto) en pixeles o porcentaje. Importante controlar la medida de la imagen

Buenas practicas

Siempre utiliza el atributo alt, evita insertar imagenes muy grandes sin reducir su tamaño y si las imagenes son responsives, utiliza porcentajes en lugar de pixelespara tamaño.

Enlaces

  • La etiqueta a se utiliza para crear enlaces a otras paginas web, documentos...

  • El elemento esencial para los enlaces es el atributo href que define la URL la cual redirige al usuario.

  • el texto entre la etiqueta de abertura y la de cierre es lo que veran los usuarios y podran clicar para ser redirigidos.

Listas

  • Listas no ordenadas

    1. Se utilizan para las listas de elementos sin un orden especifico.
    2. Los elementos se muestran con viñaetas o otros simbolos antes de cada ítem.

  • Listas ordenadas

    1. Se utilizan para listas en las que el orden de los elementos es relevante
    2. Los elementos se enumeran automaticamente (1, 2, 3...)

Atributos de las listas

  1. Atributo type (Listas ordenadas)

    Se puede utilizar para definir el tipo de enumeracion de la lista.

    • 1: Nombres
    • A: Mayúsculas
    • a: Minúsculas
    • I: Números romanos en minúsculas
    • i: Números romanos en minúsculas

  2. Atributo start (Listas ordenadas)

    Define el numero o letra para comenzar la enumeracion.

Tablas basicas

  • Etiqueta table

    La etiqueta table define la tabla y actua como el contenido principal

  • Etiqueta tr (table row)

    Define una fila dentro de la tabla

  • Etiqueta td (table data)

    Crea una celda dentro de una fila de la tabla

Atributos de tabla

  • border

    Añade un espacio a los lados de las tablas y a las celdas

  • cellpadding:

    Define el espacio entre el contenido de la celda y el espacio de los lados de la celda

  • cellspacing:

    Controla el espacio entre las celdas de la tabla

  • Etiqueta thead (table head)

    Define el encabecamiento de la tabla, agrupando las filas del encabezamiento

  • Etiqueta tbody (table body)

    Agrupa el cuerpo de la tabla, las filas de los datos. Se suele usar para separar los datos de los encabezamientos

  • Etiqueta th (table header)

    Se utiliza para crear una celda de encabezamiento en una tabla. El texto de dentro aparece en negrita y centrado por defecto
