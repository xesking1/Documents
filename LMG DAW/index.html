metadatos

dan información descriptiva sobre cómo se han de interpretar los datos del documento

la etiqueta para los metadatos es <meta>

la etiqueta <meta> siempre trabaja con alguno de los siguientes atributos:

  • charset: identifica le código de caracteres con el que se trabaja
  • name: identifica el nombre del metadato
  • http-equiv: especifica una directiva. todos los valores que ponemos aquí son encabezados http existentes
  • content: especifica el contenido que indiquen los atributos name o http-equiv
